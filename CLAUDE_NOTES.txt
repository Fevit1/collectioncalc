TECHNICAL LEARNINGS
================================================================================

JAVASCRIPT VARIABLE SCOPING:
- Cannot redeclare `let` variable with `var` even conditionally
- typeof check doesn't help - redeclaration error happens at parse time, not runtime
- Solution: Use window.variableName for globals shared across files

DATABASE MIGRATIONS:
- DBeaver excellent for running SQL migrations
- "column already exists, skipping" warnings are OK
- "does not exist, skipping" on DROP also OK
- Always verify with schema query after migration

FRONTEND/BACKEND DATA CONTRACTS:
- Always verify both sides expect same structure
- Backend expected: {items: [...]}
- Frontend sent: {title: "...", issue: "..."}
- Array wrapper vs single object is common mismatch

DEFENSIVE CODING:
- Always check if DOM elements exist before accessing: if (element) element.style = ...
- Never assume elements exist on all pages
- auth.js used by multiple pages - must be defensive

API FALLBACK STRATEGIES:
- Don't return empty/null when data missing
- Provide intelligent estimates based on available data
- Grade + Publisher + Era = reasonable estimate
- Better than $0 which breaks user experience
